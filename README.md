# Yanten快乐屋 - 微信小游戏合集 🎮

使用微信小游戏原生Canvas 2D开发的8款精选小游戏合集，支持多关卡、音效系统。

## 🎮 包含游戏

| 游戏 | 描述 | 关卡数 | 特色 |
|------|------|--------|------|
| **消消乐** | 三消宝石得高分 | 8关 | 连击加分、关卡递进 |
| **贪吃蛇** | 灵活躲避快成长 | 5关 | 速度递增挑战 |
| **2048** | 合并数字大挑战 | 5关 | 目标512→8192 |
| **方块** | 经典消除不陌生 | 5关 | 消除行数目标 |
| **飞鸟** | 穿越障碍看反应 | 5关 | 管道间隙递减 |
| **打砖块** | 击碎砖块真解压 | 5关 | 砖块数量递增 |
| **翻牌** | 记忆配对考眼力 | 5关 | 12张→36张牌 |
| **弹球** | 控制角度得高分 | 5关 | 平台宽度递减 |

## ✨ 功能特点

- 🎯 **多关卡系统** - 每个游戏5-8个难度关卡，自动保存进度
- 🔊 **音效系统** - 振动反馈模拟音效，可开关
- 🏆 **排行榜** - 本地存储Top10成绩
- 📤 **分享功能** - 分享成绩到好友
- 🎨 **主题配色** - 每个游戏独立配色风格
- 📱 **自适应布局** - 安全区域适配

## 📦 项目结构

```
mini-games/
├── game.js              # 主入口（游戏选择、切换）
├── game.json            # 游戏配置
├── project.config.json  # 微信开发者工具配置
├── common/
│   ├── config.js        # 游戏列表 + 关卡配置
│   ├── utils.js         # 公共工具（绘图、粒子、存储）
│   ├── audio.js         # 音效管理器
│   └── gameBase.js      # 游戏基类
└── games/
    ├── match3.js        # 消消乐（8关卡）
    ├── snake.js         # 贪吃蛇（5关卡）
    ├── 2048.js          # 2048（5目标）
    ├── tetris.js        # 方块（5难度）
    ├── flappy.js        # 飞鸟（5难度）
    ├── breakout.js      # 打砖块（5关卡）
    ├── memory.js        # 翻牌（5难度）
    └── bounce.js        # 弹球（5难度）
```

## 📥 快速开始

### 1. 导入微信开发者工具

1. 打开微信开发者工具
2. 选择「小游戏」项目
3. 导入本项目目录

### 2. 配置AppID

在 `project.config.json` 中修改：
```json
{
  "appid": "你的小游戏AppID"
}
```

### 3. 预览测试

点击「预览」或「真机调试」测试游戏

## 📝 开发说明

### 添加新游戏

1. 在 `games/` 创建新的游戏脚本（如 `newgame.js`）
2. 在 `common/config.js` 的 `Games` 和 `Levels` 中添加配置
3. 在 `game.js` 中导入游戏类

### 关卡配置示例

```javascript
Levels.newgame = [
  { speed: 200, name: '新手模式', target: 100 },
  { speed: 150, name: '正常模式', target: 200 },
  { speed: 100, name: '困难模式', target: 500 }
];
```
